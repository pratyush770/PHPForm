This is a basic php form

# Prequisites
1. PHP (Latest version)

# Installation
1. Clone the repo using the command: `git clone https://github.com/pratyush770/PHPForm.git`
2. Open the folder and change the directory: `cd PHPForm`
3. Open an integrated terminal and enter: `php -S localhost:8000`
4. Open any browser of your choice and enter: `http://localhost:8000/index.php`
